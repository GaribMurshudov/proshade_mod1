Standard dependencies:
======================

  This is a list of dependencies which are assumed to be already installed on the machine.
  
  This list is not exhaustive, but with these ProSHADE did install and run on clean system
  installations.

=============
Ubuntu 18.4.4 
=============

APT INSTALL:
* gcc
* g++
* gfortran
* make
* cmake
* m4
* fftw3-dev
* libblas-dev
* liblapack-dev
* swig
* python3-pip
* python3-dev
* git

PIP3 INSTALL:
* numpy

COMMAND LINE:
$ sudo apt-get update && sudo apt-get upgrade -y
$ sudo apt-get install -y gcc g++ gfortran make cmake m4 fftw3-dev libblas-dev liblapack-dev swig python3-pip python3-dev git
$ sudo pip3 install numpy

===============
Centos 8.1.1911 
===============

YUM SETUP:
sudo yum install dnf-plugins-core
sudo yum config-manager --set-enabled PowerTools

YUM INSTALL:
* gcc
* gcc-c++
* gcc-gfortran
* make
* cmake
* m4
* fftw3-devel
* blas-devel
* lapack-devel
* swig
* python3-pip
* python3-devel
* git

PIP3 INSTALL:
* numpy

COMMAND LINE:
$ sudo yum install dnf-plugins-core
$ sudo yum config-manager --set-enabled PowerTools
$ sudo yum update -y
$ sudo yum install gcc gcc-c++ gcc-gfortran make cmake m4 fftw3-devel blas-devel lapack-devel swig python3-pip python3-devel git -y
$ sudo pip3 install numpy
