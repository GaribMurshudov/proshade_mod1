Standard dependencies:
======================

  This is a list of dependencies which are assumed to be already installed on the machine.
  
  This list is not exhaustive, but with these ProSHADE did install and run on clean system
  installations.

==========
Windows 10 
==========

Not supported yet.

* Install cygwin
	* https://www.cygwin.com
* Install cygwin packages
	* Command Prompt
		* cd C:\cygwin64
		* setup-x86_64.exe -q -P wget -P gcc-g++ -P gcc-fortran -P make -P cmake -P fftw3 -P libfftw3-devel -P liblapack-devel -P swig -P zlib-devel -P git

======================
MacOS Yosemite (10.10) 
======================

* Install Command Line Tools
	* xcode-select --install
* Install Homebrew
	* ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
	* brew install gcc
	* brew install cmake
	* brew install swig
* Install pip and numpy
	* curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
	* sudo python get-pip.py
	* sudo pip install numpy
* Install numpy for python3
	* pip3 install numpy
* Install full fftw3
	* Download sources from http://www.fftw.org/download.html
	* tar -zxvf fftw-3.3.8.tar.gz
	* cd fftw-3.3.8
	* ./configure --enable-shared 
	* make
	* sudo make install

Ubuntu 18.4.4 
=============

APT INSTALL:
* gcc
* g++
* gfortran
* make
* cmake
* m4
* fftw3-dev
* libblas-dev
* liblapack-dev
* python
* python-pip
* python-dev
* python3
* python3-pip
* python3-dev
* swig
* git
* zlib1g-dev

PIP INSTALL:
* numpy

PIP3 INSTALL:
* numpy

COMMAND LINE:
$ sudo apt-get update && sudo apt-get upgrade -y
$ sudo apt-get install -y gcc g++ gfortran make cmake m4 fftw3-dev libblas-dev liblapack-dev python python-pip python-dev python3 python3-pip python3-dev swig git zlib1g-dev
$ sudo pip install numpy
$ sudo pip3 install numpy

Centos 8.1.1911 
===============

YUM SETUP:
sudo yum install dnf-plugins-core
sudo yum config-manager --set-enabled PowerTools

YUM INSTALL:
* gcc
* gcc-c++
* gcc-gfortran
* make
* cmake
* m4
* fftw3-devel
* blas-devel
* lapack-devel
* python2
* python2-pip
* python2-devel
* python3 
* python3-pip
* python3-devel
* swig
* git
* zlib-devel

PIP2 INSTALL:
* numpy

PIP3 INSTALL:
* numpy

COMMAND LINE:
$ sudo yum install dnf-plugins-core
$ sudo yum config-manager --set-enabled PowerTools
$ sudo yum update -y
$ sudo yum install gcc gcc-c++ gcc-gfortran make cmake m4 fftw3-devel blas-devel lapack-devel python2 python2-pip python2-devel python3 python3-pip python3-devel swig git zlib-devel -y
$ sudo pip2 install numpy
$ sudo pip3 install numpy
